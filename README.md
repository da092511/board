# 콘솔 게시판 
> 유저들만 이용할 수 있는 게시판 <br>

---

개시판 작성 조건

> 게시물 3회 이상 열람 시 게시물 작성 권한이 부여됨 <br>

<br>

게시판 매니저
> 반드시 첫 번째로 가입할 것 
>   * 첫 번째 가입자는 게시물 작성을 해금함 <br>
<br>

> 게시판에 게시글 하나 이상 올려 놓을 것 
>   * 게시물 열람해야 유저들이 게시글 작성 조건을 채울 수 있음 <br>


---

## 전체 메뉴


1. 회원가입

2. 로그인

3. 전체 게시물
  * 게시글 목록
    1. 이전 페이지
    2. 다음 페이지
    3. 게시물 선택
     
4. 게시물 작성

5. 마이 페이지 <br>
     1. 내 정보
     2. 내 게시물
